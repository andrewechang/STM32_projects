Project 1: LED control using button  
Function: Uses button B1 to control LED LD2, which can alternatively  
be connected to a breadboard and controlled by a switch. 

Note**: The button on the Nucleo board is active low, so the code has  
inverted logic. (reset button high = reset, reset button low = set)  

GPIO Used:  
- A5: LD2 LED
- C13: Button B1
(Note: these GPIO are the default configuration generated by STM32CubeMX)
