Project 2: LED control using rotary encoder  
Function: Uses a rotary encoder to control LED LD2, which can alternatively  
be connected to a breadboard.  

Hardware used:
- Rotary encoder module by Elegoo (datasheet attached in screenshots and documentation folder)  

GPIO Used:  
- A5: LD2 LED
- C13: Button B1
(Note: these GPIO are the default configuration generated by STM32CubeMX)  
- A0: CLK pin from encoder  
- A1: DT pin from encoder  
- A4: SW in from encoder, used as a push button to reset brightness to 50%
