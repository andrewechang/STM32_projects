# STM32_projects

This repo contains embedded systems projects developed using the STM32F446RE Nucleo board.
Each project is in its own folder and is downloadable for use with other MCUs
All projects were made in STM32CubeIDE

# Hardware Specs:
Board: STM32F446RE Nucleo-64
MCU: STM32F446RET6
Core: ARM Cortex-M4 @ 180 MHz
Flash: 512 KB
SRAM: 128 KB
